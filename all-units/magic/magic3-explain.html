The magical sum is simply the sum of all the bolded numbers!



Do you understand how the trick works?

Try playing the game again by removing rows and columns from the matrix above and see why adding all the table headers will always be the magical sum!

You can see that as you select a number and remove the row and corrosponding column you are pairing that row header with that column header to the number you selected, which is simply the sum of the two numbers! By removing the row and column you ensure that those headers will not be chosen again!
