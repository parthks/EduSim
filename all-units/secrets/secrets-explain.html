Secret Messages Decoded



Messages are encoded and decoded using linear transformations!

It all revolves around Ax = b; where A is the trasforming matrix, x is the input vector and b the ouput vector


Few things to note:

We are careful to choose an invertible matrix for A, so we can decode too!

The length of x has to the equal to the number of columns of A in order to multiply them!

All characters are converted to thier corrosponding ASCII numeric values before using them!


To decode, we repeat the process but now with the inverse of the matrix A

The reason this works is because of the property of inverses - think of the inverse of 4 => 1/4 - multiplying inverse objects always give the unit object, they "cancel" each other out.

So A-1A = I (The identity matrix)

Hence multiplying A-1 in Ax = b gives:

A-1Ax = A-1b => x = A-1b

This shows that to get back the original message we multiply the inverse of the matrix A with the encoded message!
